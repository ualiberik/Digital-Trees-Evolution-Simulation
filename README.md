# Digital Trees Evolution Simulation
## Introduction
> This project was inspired by foo52's (Simulife Hub — his english chanell) [video](https://youtu.be/IL1HogOu5B0).
"Digital Trees Evolution Simulation" project is the simulation of natural selection and evolution that shows how organisms can develop and become more complex through time. Simulation can visualize how multicellular "trees", fighting for a place in the sun, can find the most effective solutions.
Trees pass on the genome by inheritance. However, it can slightly change randomly, laucnhing changes in population and natural selection.
